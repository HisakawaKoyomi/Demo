## FlappyBird
  * canvas绘制，加载图片资源
  * ES6语法编写，面向对象思路
  * 点击中间按钮，游戏开始  
    小鸟撞到天花板/地板/铅笔，游戏结束
  * 小鸟飞过一对铅笔加一分
  * 后期可移植到微信小游戏中，移植版见（flappyBird_app）
